## 一、项目概述

### 1.1 项目定位
**智能车竞赛 - V型五连杆轮腿机器人**

本项目是一个基于**串级PID控制**的**自平衡轮腿机器人**，采用独特的"轮子管平衡，腿部管速度"解耦控制策略。

### 1.2 核心创新点
- ✨ **混合控制架构**: 舵机位控 + 无刷力控
- ✨ **解耦控制**: 直立环与速度环分离
- ✨ **VMC查表法**: 五连杆逆运动学查表实现
- ✨ **主动侧倾**: 摩托车压弯式转向
- ✨ **状态机控制**: 跳跃、排雷等复杂动作

### 1.3 控制策略总览

| 执行器 | 控制目标 | 算法 | 输出 |
|--------|---------|------|-----|
| **无刷电机 (轮子)** | 直立平衡 | 串级PID (角度环+角速度环) | PWM |
| **舵机关节 (腿部)** | 速度控制 + 姿态调整 | VMC查表 + 速度环PID | PWM |

**核心思想**:
```
轮子 → 誓死捍卫直立 (追踪机械零点)
腿部 → 通过重心变化控制速度 (前倾加速，后倾减速)
```

**功能**: 手动控制系统启停
**参数**: `state` - 0=STOP, 1=RUN
**使用场景**: 按键控制、遥控器控制

### 4.2 PID控制模块 (`pid.h`)

#### 数据结构

##### PID结构体
```c
typedef struct {
    float kp;              // 比例系数
    float ki;              // 积分系数
    float kd;              // 微分系数

    float target;          // 目标值 (设定点)
    float measure;         // 测量值 (反馈值)

    float error;           // 当前误差 = target - measure
    float last_error;      // 上次误差 (用于微分计算)
    float error_sum;       // 误差累积 (积分项)

    float output;          // PID计算输出

    float integral_max;    // 积分限幅上限 (防止积分饱和)
    float integral_min;    // 积分限幅下限
    float output_max;      // 输出限幅上限
    float output_min;      // 输出限幅下限

    uint8_t first_cycle;   // 首次循环标志 (用于微分初始化)
} pid_struct_t;
```

#### 核心函数

##### `void pid_init(pid_struct_t *pid, float kp, float ki, float kd, float integral_max, float output_max, float output_min)`
**功能**: PID控制器初始化
**参数说明**:
- `kp`: 比例系数
- `ki`: 积分系数
- `kd`: 微分系数
- `integral_max`: 积分限幅 (设为0时不限幅)
- `output_max`: 输出限幅上限
- `output_min`: 输出限幅下限

**使用示例**:
```c
pid_struct_t balance_pid;
pid_init(&balance_pid, 0.395f, 0.0f, 0.005f, 1000.0f, 10000.0f, -10000.0f);
```

##### `float pid_compute(pid_struct_t *pid, float target, float measure)`
**功能**: PID核心计算
**算法公式**:
```
Output = Kp×Error + Ki×∫Error×dt + Kd×d(Error)/dt
```

**返回值**: PID计算输出
**使用示例**:
```c
float output = pid_compute(&balance_pid, -11.05f, current_pitch);
```

##### `void pid_reset(pid_struct_t *pid)`
**功能**: 重置PID状态 (清除累积误差)
**使用场景**:
- 控制模式切换时
- 设定点大幅变化时
- 系统异常恢复后

**功能**: **核心VMC控制接口** - 根据目标高度和角度更新所有舵机
**参数**:
- `gao`: 目标高度 (单位: cm, 范围: 2.7~14.5)
- `angle`: 目标俯仰角 (单位: 度, 范围: -20~20)

**实现特性**:
- ✅ 输入限幅: 强制限制 angle 在 [-20, 20]
- ✅ 查表获取: 调用 `servo_control_tabl()` 查表
- ✅ 有效性检查: 返回10000表示超出范围
- ✅ **PWM计算**: `Target_PWM = Center + Offset × Direction`
- ✅ **防抖处理**: PWM变化率限制 (Slew Rate Limit)
- ✅ 安全限幅: 最终PWM限制在 [500, 9500]

**防抖策略详解**:
```c
#define MAX_PWM_STEP 150  // PWM每帧最大变化量

// 变化率限制逻辑:
if(target_pwm - last_pwm > MAX_PWM_STEP)
    pwm = last_pwm + MAX_PWM_STEP;  // 目标过大，只能增加MAX_PWM_STEP
else if(last_pwm - target_pwm > MAX_PWM_STEP)
    pwm = last_pwm - MAX_PWM_STEP;  // 目标过小，只能减少MAX_PWM_STEP
else
    pwm = target_pwm;               // 在允许范围内，直接使用目标值
```

**使用示例**:
```c
// 设置高度4.5cm，前倾10度
VMC_Update_All_Servos(4.5f, 10.0f);
```

**参数**:

- 角度分辨率: 0.1度 (401个点)
- 查表数据量: 394 × 401 × 2 = 315,788 个数据点

- `正常值`: 舵机偏移量 (约 -2000 ~ +2000)
- `10000`: 超出查表范围，请求无效

**使用示例**:
```c

servo_control_tabl(4.5f, 10.0f, &offset1, &offset2);
if(offset1 != 10000 && offset2 != 10000) {
    // 查表成功，计算PWM
    int32_t pwm = servo_1.center + offset1 * servo_1.dir;
    set_servo_duty(&servo_1, pwm);
}
```

## 五、控制算法详解

### 5.1 串级PID直立环

#### 控制结构
```
目标俯仰角 → [外环: 角度环PD] → 目标角速度 → [内环: 角速度环PD] → 电机PWM
                                      ↑
                              当前角速度 (陀螺仪)
                                      ↑
                        陀螺仪Y轴 → icm_output.gyro_y
```

#### 外环 - 角度环 (Angle Loop)

**参数定义**:
```c
#define ANGLE_KP   0.395f   // 比例系数
#define ANGLE_KD   0.005f   // 微分系数
#define ANGLE_KI   0.0f     // 积分系数 (一般不需要)
```

**计算公式**:
```c
float error_angle = MECHANICAL_ZERO - icm_output.pitch;  // 角度误差
float target_gyro = ANGLE_KP * error_angle              // 目标角速度
                    + ANGLE_KD * (error_angle - last_error_angle);
```

**调试建议**:
1. 从 `ANGLE_KP = 0` 开始，逐步增加
2. 直到小车能勉强维持直立
3. 过大会导致振荡，过小响应太慢

#### 内环 - 角速度环 (Gyro Loop)

**参数定义**:
```c
#define GYRO_KP    206.0f    // 比例系数
#define GYRO_KD    0.0f      // 微分系数
#define GYRO_KI    0.0f      // 积分系数
```

**计算公式**:
```c
float error_gyro = target_gyro - icm_output.gyro_y;     // 角速度误差
float pwm_output = GYRO_KP * error_gyro                 // PWM输出
                + GYRO_KD * (error_gyro - last_error_gyro);
```

**调试建议**:
1. 在角度环调好后，再调节此参数
2. 此参数影响系统的阻尼特性
3. 过大会导致高频振荡

#### 输出限幅
```c
if(pwm_output > PWM_MAX)  pwm_output = PWM_MAX;   // 10000
if(pwm_output < PWM_MIN)  pwm_output = PWM_MIN;   // -10000
```

### 5.2 异常保护与自动恢复

#### 异常保护机制

**抬头保护**:

```c
#define TAITOUMAX_TILT  30.0f
```

**恢复条件**:
```c
#define RECOVERY_THRESHOLD  5.0f
#define RECOVERY_TIME       2000  // ms

// 角度回到正常范围内，并持续2秒
if(ABS(icm_output.pitch - MECHANICAL_ZERO) < RECOVERY_THRESHOLD) {
    if(recovery_counter++ > RECOVERY_TIME) {
        g_System_State = SYSTEM_STATE_RUN;  // 自动恢复运行
    }
}
```

### 5.3 速度控制策略 (腿部控制)

**核心思想**: 通过改变机身重心来"诱骗"轮子加速或减速

**控制逻辑**:
```
目标速度 → [速度环PID] → 目标俯仰角 → [VMC查表] → 舵机偏移量 → 重心变化 → 轮子加速/减速
```

**控制映射**:
- **前倾 (+Pitch)** → 重心前移 → 轮子向前加速
- **后仰 (-Pitch)** → 重心后移 → 轮子向前减速或向后加速
- **侧倾 (+Roll)** → 左腿缩短、右腿伸长 → 左转时压弯

---

### 5.4 主动侧倾控制 (摩托车压弯)

**应用场景**: 高速过弯、防止翻车

**控制公式**:
```c
float turn_output = GPS_Error + Vision_Error - Kd * Gyro_Z;      // 转向输出
float target_roll = K_Bank * turn_output * current_speed;        // 目标侧倾角
```

**实现逻辑**:
```c
// 左转时：
```

### 8.3 常见问题排查

#### 问题1: 小车无法直立
**可能原因**:
- 机械零点设置错误
- ANGLE_KP 过小
- 陀螺仪方向反了

**解决方法**:
1. 重新测定机械零点
2. 逐步增大 ANGLE_KP
3. 检查 `icm_output.gyro_y` 符号

#### 问题2: 小车高频振荡
**可能原因**:
- ANGLE_KP 或 GYRO_KP 过大
- 采样周期不稳定
- 机械间隙过大

**解决方法**:
1. 减小 ANGLE_KP 和 GYRO_KP
2. 检查定时器中断优先级
3. 紧固机械连接

#### 问题3: 小车缓慢倒下
**可能原因**:
- GYRO_KP 过小
- 积分未清零导致累积误差

**解决方法**:
1. 增大 GYRO_KP
2. 调用 `pid_reset()` 清除累积误差
