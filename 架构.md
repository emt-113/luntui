### 1.2 控制策略总览

#### 解耦控制策略

本系统采用独特的**"轮子管平衡，腿部管速度"**解耦控制策略:

| 控制目标 | 执行器 | 控制算法 | 输出 | 响应速度 |
|---------|-------|---------|-----|---------|
| **直立平衡** | 无刷电机(轮子) | 串级PID (角度环+角速度环) | PWM | 极快 (1ms) |
| **速度控制** | 舵机(腿部) | 速度环PID + VMC查表 | PWM角度 | 较快 (20ms) |
| **转向控制** | 双轮差速 | 差速控制算法 | 左右PWM差 | 快 (1ms) |
| **侧倾控制** | 舵机(左右腿) | 主动侧倾算法 | 高度差 | 较快 (10ms) |

**核心思想**:
```
轮子 → 誓死捍卫直立 (追踪机械零点 -11.05°)
腿部 → 通过重心变化控制速度 (前倾加速，后倾减速)
```


## 五、核心算法架构

### 5.1 PID控制器架构

```c
// pid.h - 通用PID结构体
typedef struct {
    float kp;              // 比例系数
    float ki;              // 积分系数
    float kd;              // 微分系数

    float target;          // 目标值
    float measure;         // 测量值

    float error;           // 当前误差
    float last_error;      // 上次误差
    float error_sum;       // 误差累积

    float output;          // PID输出

    float integral_max;    // 积分限幅
    float integral_min;
    float output_max;      // 输出限幅
    float output_min;

    uint8_t first_cycle;   // 首次循环标志
} pid_struct_t;
```

**PID计算流程**:
```
1. 计算误差: error = target - measure
2. 积分累积: error_sum += error
3. 积分限幅: if (error_sum > integral_max) error_sum = integral_max
4. 微分计算: d_error = error - last_error
5. PID输出: output = Kp*error + Ki*error_sum + Kd*d_error
6. 输出限幅: if (output > output_max) output = output_max
7. 更新历史: last_error = error
```

### 5.2 VMC查表架构

```c
// vmc_pwm_table.h - 五连杆逆运动学查表
// 维度: [高度索引][角度索引][舵机编号]

// 高度范围: 2.7cm ~ 14.5cm (394个点, 分辨率0.03cm)
// 角度范围: -20° ~ +20° (401个点, 分辨率0.1°)
// 舵机数量: 2个 (offset1, offset2)

// 查表函数:
void servo_control_tabl(
    float p,           // 目标高度 (2.7~14.5 cm)
    float angle,       // 目标角度 (-20~20 度)
    int16* pwm1,       // 输出: 舵机1/3偏移量
    int16* pwm2        // 输出: 舵机2/4偏移量
);

// 返回值:
//   - 正常值: 舵机偏移量 (~ -2000~+2000)
//   - 10000:  超出查表范围
```

**PWM计算公式**:
```c
Target_PWM = Center + Offset × Direction

// 示例 (Servo_1: 左前腿)
int32_t target_pwm = servo_1.center + offset1 * servo_1.dir;
//                 4650         + offset1 * (-1)
```

**防抖处理**:
```c
#define MAX_PWM_STEP 150

if (target_pwm - last_pwm > MAX_PWM_STEP)
    pwm = last_pwm + MAX_PWM_STEP;
else if (last_pwm - target_pwm > MAX_PWM_STEP)
    pwm = last_pwm - MAX_PWM_STEP;
else
    pwm = target_pwm;
```

| **moter_task()** | 1ms | 高 | moter.c | 电机PWM输出更新 |
| **ICM_getValues()** | 5ms | 高 | filt.c | IMU数据读取 |
| **uart_task()** | 10ms | 中 | uart.c | 串口数据收发 |
| **VMC_Update_All_Servos()** | 20ms | 中 | Servo.c | 舵机VMC控制更新 |

### 6.3 中断优先级分配

```

   7    | PIT定时器 (1ms)          | control_run_1ms()

   5    | SPI/I2C (ICM20602)      | IMU数据读取
   4    | PWM定时器               | 舵机PWM输出
   3    | UART1 (调试串口)        | 调试数据输出
   2    | GPIO中断 (按键)         | 按键控制
```


### 7.2 核心文件说明

#### **control.c/.h** (核心控制层)
- **功能**: 直立环控制、速度环控制、异常保护
- **关键函数**:
  - `control_init()` - 控制器初始化
  - `control_run_1ms()` - 1ms核心控制任务
  - `control_set_state()` - 系统状态控制
  - `speed_control_run()` - 20ms速度环控制
- **依赖模块**: pid, filt, Servo, moter, uart
- **全局变量**:
  - `g_System_State` - 系统状态 (STOP/RUN)
  - `g_Debug_Pitch` - 当前俯仰角
  - `g_Debug_Gyro` - 当前角速度
  - `g_Debug_PWM` - 最终PWM输出
  - `g_Debug_Target_Speed` - 速度环目标速度
  - `g_Debug_Measure_Speed` - 速度环测量速度
  - `g_Debug_Speed_Output` - 速度环PID输出

#### **pid.c/.h** (PID算法库)
- **功能**: 通用PID控制算法
- **关键函数**:
  - `pid_init()` - PID初始化
  - `pid_compute()` - PID核心计算
  - `pid_reset()` - 重置PID状态
- **特点**: 支持积分限幅、输出限幅、通用性强

#### **Servo.c/.h** (舵机控制 + VMC)
- **功能**: 舵机PWM驱动、VMC五连杆控制
- **关键函数**:
  - `Servo_Init()` - 舵机初始化
  - `set_servo_duty()` - 设置舵机PWM
  - `VMC_Update_All_Servos()` - VMC控制接口 ⭐
- **依赖**: vmc_pwm_table (查表数据)

#### **filt.c/.h** (姿态解算)
- **功能**: ICM20602数据读取、姿态解算
- **关键函数**:
  - `ICM_getValues()` - 读取IMU原始数据
  - `ICM_getEulerianAngles()` - 姿态角解算
  - `gyroOffset_init()` - 陀螺仪零点校准
- **输出**: `icm_output` 结构体 (pitch, roll, yaw, gyro_x/y/z)

#### **vmc_pwm_table.c/.h** (五连杆查表)
- **功能**: 五连杆逆运动学查表数据
- **数据量**: 394 × 401 × 2 = 315,788 个数据点
- **关键函数**:
  - `servo_control_tabl()` - 查表核心函数
- **特点**: 纯查表,无实时计算,速度快
